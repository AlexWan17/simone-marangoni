# Pushing this Project to GitHub

Follow these steps to upload the contents of this project to the repository `https://github.com/AlexWan17/zoone-mvp-final`.

1. Initialize the remote if it is not already set:
   ```bash
   git remote add origin https://github.com/AlexWan17/zoone-mvp-final.git
   ```
   Replace `origin` with a different name if a remote with that name already exists.

2. Verify that all local work is committed:
   ```bash
   git status
   ```

3. Push the current branch to the GitHub repository:
   ```bash
   git push -u origin main
   ```
   If your default branch has a different name, substitute `main` with that branch.

Make sure you have the necessary permissions to push to the GitHub repository and that your SSH or HTTPS credentials are configured correctly.
